### Hi, I'm Sebastián

Senior fullstack engineer at **Gorilla Logic**, based in Cali, Colombia. I build across web, mobile and games — and I write about what I learn along the way at **[valdemird.com](https://valdemird.com)**.

Lately I'm focused on **AI-assisted development** (Claude Code, spec-driven workflows), **testing as a control surface**, and **local generative AI** — training and orchestrating diffusion models to produce game assets on my own hardware.

> I build to learn. If I can't build it, I don't really understand it.
